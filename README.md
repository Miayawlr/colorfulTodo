#### 极简 react todo app V1

> 一个使用 hooks 搭建的 简单 demo

### 如何使用

选择 V1 分支进行拉取
这是[项目地址](https://github.com/Miayawlr/colorfulTodo)的源码。
你也可以使用此项目自己 DIY 学习。

```shell
# 安装依赖
npm install / yarn add

# 先运行服务！！！
yarn server / npm run server
#启动项目
yarn start /npm run start
```
